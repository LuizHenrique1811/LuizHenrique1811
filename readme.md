## ⚙️ Arquitetura & Stack Tecnológica

O agrupamento abaixo reflete minha atuação em diferentes camadas sistêmicas:

- **Core & Rendering:** Next.js, React, TypeScript.
- **Garantia de Qualidade (QA) & Automação:** Desenvolvimento orientado a comportamento (BDD) utilizando Gherkin, testes com Pytest e automação de pipelines de CI/CD via GitHub Actions.
- **Dados & Integrações:** Modelagem relacional (DrawSQL), Caching e mensageria (Redis), orquestração de webhooks (ex: monitoramento de serviços via Slack) e consumo de APIs transacionais (Mercado Pago).
- **Sustentação & Refatoração:** Manutenção, engenharia reversa e modernização de dívida técnica em sistemas legados (ASP Classic).
- **Design & Orquestração:** Figma, Vercel.
